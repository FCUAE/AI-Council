3. Component Architecture (The Refactor Plan)
We are not building new logic; we are wrapping existing logic in better components.

<CouncilChamber />: The new main wrapper. It checks your existing status variable to determine if it shows the Grid (Hearing) or the Hero Card (Verdict).

<CouncilSeat />: Replaces your current "Model Response" div. It handles the "streaming" visual style.

<ChairmanCard />: The new home for the final answer.

<EvidenceVault />: The tabs component that holds the rest of the data.
