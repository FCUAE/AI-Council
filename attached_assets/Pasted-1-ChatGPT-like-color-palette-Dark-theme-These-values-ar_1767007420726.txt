1. ChatGPT like color palette (Dark theme)

These values are approximations of what you see in the interface, chosen to feel very similar without copying anything internal.

:root {
  /* Base background and surfaces */
  --cgpt-bg: #050509;
  --cgpt-bg-alt: #0c0c13;
  --cgpt-panel: #101017;
  --cgpt-panel-alt: #151521;

  /* Borders and subtle outlines */
  --cgpt-border-subtle: #252535;
  --cgpt-border-strong: #3a3a4d;

  /* Text colors */
  --cgpt-text-primary: #f9fafb;
  --cgpt-text-secondary: #c7cad4;
  --cgpt-text-muted: #9ca0b3;
  --cgpt-text-inverse: #050509;
  --cgpt-text-danger: #fecaca;

  --cgpt-accent-soft: rgba(16, 163, 127, 0.12);

  /* Helpful semantic colors */

  --cgpt-warning: #facc15;

2. Optional light theme variant

If you want a toggle later, you can define a .theme-light palette.

.theme-light {
  --cgpt-bg: #ffffff;
  --cgpt-bg-alt: #f4f4f6;
  --cgpt-panel: #f9fafb;
  --cgpt-panel-alt: #eef0f5;

  --cgpt-border-subtle: #d2d6e0;
  --cgpt-border-strong: #9ca3af;

  --cgpt-text-primary: #0f172a;
  --cgpt-text-secondary: #4b5563;
  --cgpt-text-muted: #6b7280;
  --cgpt-text-inverse: #ffffff;

  --cgpt-accent: #10a37f;
  --cgpt-accent-soft: rgba(16, 163, 127, 0.08);
  --cgpt-accent-strong: #067a5b;

  --cgpt-msg-user-bg: #e5e7eb;
  --cgpt-msg-assistant-bg: #f9fafb;
  --cgpt-msg-system-bg: #e5f2ec;

  --cgpt-input-bg: #ffffff;
  --cgpt-input-border: #d1d5db;
  --cgpt-input-placeholder: #9ca3af;

  --cgpt-scrollbar-thumb: #cbd5f5;
  --cgpt-selection-bg: rgba(16, 163, 127, 0.25);
}


Apply with something like:

<body className="theme-light">...</body>


or swap classes in your React root.

4. How to use this in your React app

Create a CSS file, for example chatgpt-theme.css, and paste:

The :root variables

The layout and chat styles you like

Import it in your app entry:

import "./chatgpt-theme.css";


Use class names:

<div className="cgpt-app">
  <aside className="cgpt-sidebar">...</aside>
  <main className="cgpt-main">
    <div className="cgpt-chat">
      <div className="cgpt-chat-inner">
        <div className="cgpt-msg cgpt-msg-user">
          <div className="cgpt-avatar cgpt-avatar-user">U</div>
          <div className="cgpt-msg-body">Hi</div>
        </div>
        <div className="cgpt-msg cgpt-msg-assistant">
          <div className="cgpt-avatar cgpt-avatar-assistant">AI</div>
          <div className="cgpt-msg-body">Hello</div>
        </div>
      </div>
    </div>

    <div className="cgpt-input-wrapper">
      <div className="cgpt-input-inner">
        <div className="cgpt-input">
          <textarea placeholder="Send a message" />
          <button className="cgpt-send-btn">➤</button>
        </div>
      </div>
    </div>
  </main>
</div>
